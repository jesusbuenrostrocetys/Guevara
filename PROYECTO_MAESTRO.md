# SYSEA – Sistema de Inteligencia Comercial para Radiadores

## 📋 INFORMACIÓN GENERAL DEL PROYECTO

**Empresa:** SYSEA Radiadores  
**Giro:** Venta de radiadores automotrices  
**Propietario:** Mario Guevara  
**Sistema:** Excel 365 + VBA  
**Estado:** En desarrollo por Sprints

---

## 🎯 OBJETIVO GENERAL

Construir un sistema integral en Microsoft Excel 365 que administre completamente la operación comercial de SYSEA Radiadores, enfocado en:

✅ Maximizar utilidad  
✅ Mantener competitividad  
✅ Fidelizar talleres  
✅ Evitar sobreinventario  
✅ Facilitar operación diaria  
✅ Proteger márgenes mínimos  
✅ Reducir errores humanos  
✅ Generar información para decisiones

---

## 🏗️ ESTRUCTURA DEL PROYECTO

El libro de Excel contendrá:

1. **Catálogo Maestro** - Base de datos de productos
2. **Entradas** - Registro de compras a proveedores
3. **Ventas POS** - Punto de venta
4. **Tickets** - Recibos generados
5. **Dashboard** - Indicadores gerenciales
6. **Reabastecimiento** - Sugerencias de compra
7. **Alertas y Configuración** - Control y parámetros
